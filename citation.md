# Citation

Researchers using the Comprehensive Courts of Appeals Database (CCAD) should cite both the methods paper and the specific data release.

## Methods Paper

Kastellec, Jonathan P., and Eitan Sapiro-Gheiler. 2026. "Appealing to Large-Language-Model-as-Judge: A Comprehensive Machine-Coded Database for the U.S. Courts of Appeals."

```bibtex
@article{kastellec_sapirogheiler_ccad_methods,
  author = {Kastellec, Jonathan P. and Sapiro-Gheiler, Eitan},
  title = {Appealing to Large-Language-Model-as-Judge: A Comprehensive Machine-Coded Database for the U.S. Courts of Appeals},
  year = {2026}
}
```

## Dataset

Sapiro-Gheiler, Eitan, and Jonathan P. Kastellec. 2026. "Comprehensive Courts of Appeals Database (CCAD)." Harvard Dataverse, V1. https://doi.org/10.7910/DVN/XN2FVS.

```bibtex
@misc{sapiro-gheiler_kastellec_ccad,
  author    = {Sapiro-Gheiler, Eitan and Kastellec, Jonathan P.},
  title     = {{Comprehensive Courts of Appeals Database (CCAD)}},
  year      = {2026},
  publisher = {Harvard Dataverse},
  version   = {V1},
  doi       = {10.7910/DVN/XN2FVS},
  url       = {https://doi.org/10.7910/DVN/XN2FVS}
}
```
